# <h1> <i> <mark> 💬 ChatViz </mark> </i> </h1>

### Transform WhatsApp Conversations into Actionable Insights

> **Upload a WhatsApp chat export and uncover meaningful statistics, activity trends, word clouds, user engagement, and much more—all through an interactive Streamlit dashboard.**


## 🚀 Why ChatViz?

WhatsApp conversations contain thousands of messages that are difficult to interpret manually.

ChatViz transforms raw chat exports into meaningful visual insights, helping users understand communication patterns, activity trends, participation levels, and language usage through an intuitive interactive dashboard.

## ⚙️ Tech Stack
| Layer           | Technology |
| --------------- | ---------- |
| Frontend        | Streamlit  |
| Backend         | Python     |
| Data Processing | Pandas     |
| Visualization   | Matplotlib |
| Word Cloud      | WordCloud  |

## 📦 Installation

```bash
git clone https://github.com/sanskarhere/ChatViz.git
cd ChatViz
pip install -r requirements.txt
python -m streamlit run app.py
```
